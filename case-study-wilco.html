The Challenge

The client wanted to modernize the website's appearance while maintaining brand consistency. Key goals included integrating visual elements from the logo, such as the wave pattern and fire-and-ice color scheme throughout the site, and incorporating a branded truck image into the design. The challenge was to achieve these updates using only CSS and HTML, without modifying the base theme's HTML structure.

Color Usage

  • Fire colors (#E94E1B & #FF6B35): Buttons, CTAs, urgent/action elements
  • Ice colors (#0066CC, #0080E8, #4DA6FF): Links, hover states, trust/professional elements
  • Neutrals (#1A2332 & #2C3E50): Text, backgrounds, structure

Typography

Fonts Used:

  • Raleway (Bold) for headings - More elegant and refined
  • Lato for body text - Similar to Open Sans but slightly warmer

Live Typography Preview

Water and Fire Damage Recovery

Lato provides excellent readability for body text with a warm, approachable feel. It ensures users can easily consume information about restoration services, emergency response, and project details.

View CSS Code 
:root {
  --fire-primary: #5C1818;
  --fire-secondary: #D32F2F;
  --ice-primary: #006B7D;
  --ice-secondary: #1E3A5F;
}

.btn {
  background: linear-gradient(135deg, var(--fire-primary) 0%, var(--fire-secondary) 100%);
  border: none;
  border-radius: 50px;
  padding: 14px 32px;
  color: white;
  font-weight: 600;
  text-transform: uppercase;
  letter-spacing: 0.5px;
  transition: all 0.3s ease;
  position: relative;
  overflow: hidden;
  box-shadow: 0 4px 15px rgba(233, 78, 27, 0.3);
}

.btn::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(135deg, var(--ice-primary) 0%, var(--ice-secondary) 100%);
  transition: left 0.4s ease;
  z-index: 0;
}

.btn:hover::before {
  left: 0;
}

.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 20px rgba(0, 102, 204, 0.4);
}
